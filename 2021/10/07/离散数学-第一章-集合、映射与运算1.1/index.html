集合、映射与运算

集合的有关概念

集合的元素

  1. 通常用大写的英文字母如A、B、C、D表示集合，小写的字母如a、b、c、d表示集合中的元素
  2. 集合A中的元素个数（又称基数或势）为n用 |A|=n 表示
  3. 元素与集合关系为 a∈A 或 a∉A

集合的表示方法

  1. 列举法

    A = {1,2,3,4,5……}

  2. 描述法

    A = {x|x>=10且x<15}

  3. 迭代法（又称归纳法）

    n∈N,有限次数进行n+1仍有n+1∈N

  4. 文氏图






集合的特征

  1. 确定性

    要有明确尺度，不能用个子高等语言描述，否则称为模糊集合

  2. 互异性

    记集合A={1,2}，B={1,1,2,1}，则A=B且|A|=2

  3. 无序性

    记集合A={1,2}，B={2,1}，则A=B

  4. 多样性
    集合内部的元素是独立的，如集合A = {a,{a},{a,d,c},地球}

特殊集合

  1. 包含是否有限个元素的集合分别称为有限（有穷）集和无限（无穷）集

  2. 约定俗称的集合：

    • 自然数集 ： N = {0,1,2……}
    • 整数集 : I = {……-2,-1,0,1,2……}
    • 正整数集 ： I+ = {1,2,3……}
    • 负整数集 ： I- = {-1,-2,-3……}
    • 素数集 ： P = {2,3,5,7,11,13……}
    • 偶数集合 ： Ev = {……-4,-2,0,2,4……}
    • 奇数集合 ： Od = {……-3,-1,1,3……}
    • 实数集合 ： R
    • 复数集合 : C
    • 有理数集合 ： Q

  1. 空集

    ∅和{∅}是不一样的，∅∈{∅}，且∅是任一集合的子集，空集是唯一的

  2. 全集

    研究对象需定义全集，通常用E或U表示

  3. 补集

    定义 E = I- ，则补集 ~E = N

  4. 子集

    子集和真子集分别用⊆和⊂表示，其平凡子集为空集和其自身

  5. 幂集

    设有集合A，A的所有子集构成的集合为A的幂集，记作 P(A）或 2A

    P(A) = {S|S ⊆ A}

    特别地，P(∅) = {∅}

集合的运算

  1. 交集

    S = A ∩ B ={x|x ∈ A ^ X ∈ B}

  2. 并集

    S = A U B = {x|x ∈ A V x ∈ B}

  3. 相对补集

    S = A - B = A ∩－B＝{x|x ∈ A ^ x ∉ B}

  4. 对称差集

    S = A Δ B = A ⊕ B = (A-B)U(B-A)

  5. 极小项性质

    若A ∩ B = ∅，则A ⊕ B = A U B

  6. 集合恒等式

    • 幂等性 A U A=A A∩A=A
    • 交换律 U∩⊕均适用
    • 结合律 U∩⊕均适用
    • 分配律 U∩-均适用
    • 同一律（幺元律） A U ∅ = ∅   A ∩ E = A  A - ∅ = A   A ⊕ ∅ = A
    • 零元律 A U E = E A ∩ ∅ = ∅
    • 互补律 A Ｕ～A＝E　A∩ ～A ＝∅
    • 吸收律 A U (A ∩ B)＝A　A ∩ (A U B)＝A　
    • 德摩根律 -（A U B) = -A ∩ -B    A - ( B ∩ C ) = (A - B) U (A - C)

  7. 特殊关系

    A ⊕ B = （A U B) - (A ∩ B) = (A∩-B)U(-A∩B) = (A-B)U(B-A)

  8. 集合恒等式的证明方法

    基本定义法（从元素出发）、文氏图法、公式推演法

计数问题

  1. 加法原理和乘法原理

    • 加法原理

    ​ 若有n个集合A1到An，且Ai∩Aj=∅，i、j∈{1,2,……n}且i != j，则 |(A1 U A2 U……UAi)| = |A1|+|A2|+……+|An|

    • 乘法原理

    ​ 若一个事件要n步完成，第i步有mi个方案，则完成改事件的总方案数有 |A1|*|A2|*……*|An|

  2. 排列与组合

    • 排列

    从含n个元素的集合A中有序选择r个元素称为A的一个r排列，记作P(n,r) = n*(n-1)*……*(n-r+1) = n! / (n-r)!

    • 组合

    从含n个元素的集合A中无序选择r个元素称为A的一个r组合，记作C(n,r) = n*(n-1)*……*(n-r+1) / r! = n! / [( n - r )! * r!]

  3. 常用结论

    • |A1UA2|<=|A1|+|A2|
    • |A1∩A2|<=min(|A1|,|A2|)
    • |A1-A2|>=|A1|-|A2|
    • |A ⊕ B| = |A1|+|A2| - 2|A1∩A2|
    • |A1UA2| = |A1|+|A2| - |A1∩A2| (容斥原理)
    • |A U B U C| = |A| + |B| +|C| - |A∩B| - |A∩C|-|B∩C| + |A∩B∩C| (容斥原理推广，奇加偶减)

集合的划分

  1. 划分的定义

    对于非空集合A，假设|A| = m，若有集合簇 π ={A1，A2，……，An}

    使得：

    ​ Ai != ∅ (i=1,2,3,……,n)

    ​ Ai ⊆ A (i=1,2,3,……,n)

    ​ Ai ∩ Aj = ∅ (i,j=1,2,3,……,n且i!=j)

    ​ A1 U A2 U A3 U……U An = A

    称 π 为集合A的一个划分，元素Ai称为划分块

  2. 最大划分和最小划分

    对于上述集合簇 π ，若|π|=m，则称 π 为集合A的最大划分，若|π| = 1，则称π为集合A的最小划分

  3. 加细（细分）与真加细

    此概念是在划分的基础上进行比较，举例如下：

    记A={a,b,c} π1={{a},{b},{c}} π2={{a},{b,c}}

    划分自己是自己的加细；

    π1是π2的真加细，即π1中每个划分块里的元素都能在π2中某个划分块里的找到

  4. 交叉划分

    A={a,b,c} π1={{a,b},{c}} π2={{a},{b,c}}

  5. 划分的总数

    假设集合|B| = n ，则集合A的所有划分的个数N(B) = S(n,0)+S(n,1)+……+S(n,n)

    其中 S(n,k) 表示把含n个元素的集合分成k块的总方案数

    有N(B0)=1,N(B1)=1,N(B2)=2,N(B3)=5,N(B4)=15,N(B5)=52,N(B6)=203

    其中简写N4 = C(3,0)B0+C(3,1)B1+C(3,2)B2+C(3,3)B3 = 1x1+3x1+3x2+5x1 = 15

集合的覆盖

  1. 覆盖的定义

    对于非空集合A，假设|A| = m，若有集合簇 π ={A1，A2，……，An}

    使得：

    ​ Ai != ∅ (i=1,2,3,……,n)

    ​ Ai ⊆ A (i=1,2,3,……,n)

    ​ A1 U A2 U A3 U……U An = A

    称 π 为集合A的一个覆盖，覆盖与划分唯一区别为划分有Ai ∩ Aj = ∅ (i,j=1,2,3,……,n且i!=j)，即划分块不能重复，

    而覆盖可以

  2. 完全覆盖

    划分块Ai间不存在子集关系称为集合A的一个完全覆盖

    例如 A = {a,b,c} B = {{a},{a,b},{c}} C = {{a,b},{b,c}}

    B是A的覆盖，因{a}⊆{a,b}所以不是A的完全覆盖

    C是A的完全覆盖
