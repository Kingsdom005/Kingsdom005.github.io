序偶与笛卡尔乘积

  1. 序偶的定义

    由两个给定次序的x和y组成的序列称为序偶（二元组，可推广到n元组），记作或（x,y)，在第i个位置称为第i分量

  2. 序偶的性质

    两序偶相等，所有分量必须一一对应相等

    有序n元组指由n个给定次序的个体a1到an组成的序列

  3. 笛卡尔乘积

    定义为：设A，B是任意两个集合，取A中元素作为第一元，B中元素作为第二元所构成的所有序偶的集合，

    称为A和B的笛卡尔乘积（又称卡氏积或直积），记作：

    A x B = { < x , y > | x ∈ A , y ∈ B }

    若|A| = m，|B| = n，则|A x B| = m x n

    特别地，空集的基数取0 ，即A x ∅ = ∅

  4. 笛卡尔积的性质

    • 满足分配律，不满足交换律
    • 若C ≠ ∅ ，则 A ⊆ B ⇔ (AxC)⊆(BxC) ⇔ (CxA) ⊆ (CxB)
    • 若A，B，C，D四个集合非空，则AxB⊆CxD当且仅当A⊆C且B⊆D

映射与函数

  1. 定义

    设A，B是任意给定的两个集合，f是从A到B的一个对应法则，对于∀ x ∈ A，∃唯一y ∈ B与之对应，记作< x,y > ∈ f，

    则称f是从A到B的一个映射

    对< x,y > ∈ f，x为像源或自变量，f的定义域记作dom(f)=A，y为f作用在x下的像或函数值，一般用y=f(x)代替< x,y > ∈ f，

    至于ran(f) = y，且ran(f)={ y | y ∈ B ^ ∃x (x ∈ A ^ y = f(x) ) }

  2. 构成映射或函数的条件

    定义域内的所有自变量都有唯一对应的一个函数值

    特别地，若给定两个集合A，B，函数< A,B > ∈ f，则

    • 定义域为集合A，而不是A的子集
    • 函数的值域ranf( A )是集合B的子集，即f( A ) ⊆ B
    • 函数f的基数等于定义域的基数，即|f(A)|=|B|
    • f(x)表示一个函数值，而f是一个函数，表示一个序偶的集合，f( x ) ≠ f

  3. 集合间的映射

    设有限集A，B的基分别为m,n，则从A到B共有nm个不同的映射

    区别笛卡尔乘积的基！！！

  4. 举例分析：记A={a,b,c} B={1,2}

    笛卡尔乘积个数：每个元素有两种选择，选择一次，共3x2=6种，列举如下：

    < a,1 >,< a,2 >,< b,1 >,< b,2 >,< c,1 >,< c,2 >

    A、B集合间映射个数为：23=8种，列举如下：

    f0={ < a,1 >,< b,1 >,< c,1 > }

    f1={ < a,1 >,< b,1 >,< c,2 > }

    f2={ < a,1 >,< b,2 >,< c,1 > }

    f3={ < a,1 >,< b,2 >,< c,2 > }

    f4={ < a,2 >,< b,1 >,< c,1 > }

    f5={ < a,2 >,< b,1 >,< c,2 > }

    f6={ < a,2 >,< b,2 >,< c,1 > }

    f7={ < a,2 >,< b,2 >,< c,2 > }

    很容易发现，从A映射到B，每个f里有|A|个位置，每个位置有|B|种选择。

    并且，f0 U f1 U …… U f7 = { ,,,,, }

  5. 单射、满射、双射函数

    单射：函数一个x对应一个y，可以多个x对应一个y；单射函数只能一个x对应一个y

    满射：满足|A|=|f(A)|=|B|

    双射：满足单射和满射，存在逆函数

  6. 置换

    若A是有限集合，将A到A的双射称为A上的置换（2x3矩阵形式）
