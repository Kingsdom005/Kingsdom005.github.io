# Programmer Resume Website

A programmer-oriented personal portfolio site for interview showcase and GitHub Pages deployment.

## Included files

- `index.html` - Main page with full content
- `styles.css` - Flat and clean visual design
- `script.js` - Mobile menu interaction
- `README.md` - Setup instructions

## Deployment

1. Upload all files to your `Kingsdom005.github.io` repository root
2. Commit and push to `main`
3. Open repository settings
4. Go to `Pages`
5. Select:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/(root)`

## What to customize next

- Replace education information with real school and major
- Replace projects with your actual repositories and achievements
- Add a resume PDF link if needed
- Optionally add project screenshots
